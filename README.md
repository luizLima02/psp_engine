# psp_engine
A psp engine with the principals functionalities made.
The engines uses Data Oriented Design in most parts of the engine, with the exception of parts where it could cause perfomances decrease.
# dependences
### PSPSDK
is required to have the [pspsdk](https://pspdev.github.io/) intalled to be able to compile the files.
### Make
is required to compile the project.
